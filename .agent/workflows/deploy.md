# Deploy to GAS

## Description
ローカルの変更をGAS環境にアップロードし、新しいバージョンとして公開デプロイを行います。

## Context
ユーザーから「デプロイして」「更新を反映して」と指示された場合に実行します。

## Steps
1. ターミナルで `clasp status` を実行し、設定ファイルの有効性を確認する。
2. `clasp push -f` を実行して、ローカルのコードをGASプロジェクトへ強制アップロードする（競合回避のため）。
3. `clasp deploy` を実行し、新しいバージョンを発行する。
4. 実行結果を確認し、成功した場合は「デプロイが完了しました」と報告する。エラーが出た場合はログを表示して対策を提案する。
