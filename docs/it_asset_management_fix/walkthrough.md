# 修正内容の確認：IT資産管理画面

## 1. 修正の検証項目

### A. データ未受信時の挙動
- 「IT資産管理」タブを開き、機器リストが空の場合に「【重要】データが0件です...」という赤い警告が表示されることを確認。
- データが存在する場合、青い文字で「データ受信確認: N 台の機器」と表示されることを確認。

### B. レイアウト配置（上詰め）
- 機器リスト（カード表示）が画面上端から始まっていることを確認。
- ネットワーク構成図タブにおいて、図がコンテナの上端に密着し、不自然な空白（Vertical Center）がないことを確認。

## 2. コード変更点
- **CSS**: `.it-assets-view-wrapper`, `.network-map-container`, `#networkMap svg` を更新。
- **HTML**: `v-if/v-else` デバッグ行を追加、共通ラッパーを `it-assets-view-wrapper` に変更。
- **JS**: 描画ターゲットを `document.getElementById('networkMap')` に修正。

## 3. 次のステップ
- 本番の `Code.gs` からデータが正しく返ってこない場合は、スプレッドシートのシート名や権限設定を確認してください。
